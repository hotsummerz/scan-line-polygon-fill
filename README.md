Implementasi algoritma Scan Line Polygon Fill menggunakan Python dan Tkinter

Cara run:
1. Buka new terminal
2. ketik python scanlinepolygonfill.py
3. Klik animasi untuk melihat animasi scan line merah berjalan dari atas ke bawah dan titik interseksi muncul. Klik langsung untuk menampilkan hasil fill tanpa animasi
4. Klik reset untuk kembali ke polygon kosong
